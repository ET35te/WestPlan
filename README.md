## 📖 项目简介

这是一个**卡牌 + 叙事** 的独立游戏，玩家扮演汉代使者，沿丝绸之路向西行进。

### 核心玩法
- 🎴 **卡牌系统**: 出牌进行战斗
- 📖 **事件系统**: 随机事件影响游程  
- 🗺️ **地图推进**: 12 个节点 + 终局
- 💾 **存档系统**: 支持游戏进度保存

## 🔧 快速开始

### 5 分钟体验

```bash
1. 打开 Unity 项目
2. Play 运行游戏
3. 点击 "开始游戏"
4. 看到事件 ✅
5. 进入战斗 ✅
6. 看到红色飘字 ✅
```

### Debug 快捷键

| 快捷键 | 功能 |
|--------|------|
| **Ctrl+I** | 资源无限 |
| **Ctrl+K** | 秒杀敌人 |
| **Ctrl+W** | 快速胜利 |
| **Ctrl+D** | 打印资源 |
| **Ctrl+B** | 打印战斗 |
